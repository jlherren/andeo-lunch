# Lunch money frontend

## Install dependencies

```
pnpm install
```

## Serve the app with hot-reloads for development

```
pnpm run serve
```

## Compile for production

Copy `.env` to `.env.local` and make adjustments for your setup.


```
pnpm run build
```

## Run tests

```
pnpm run test
```

## Lints and fixes files

```
pnpm run lint
```
